Cara pakai setiap fitur — mudah dipahami, langkah demi langkah

Selamat datang di Social Media Marketing Agent! 🎉

Ini adalah aplikasi yang bisa otomatis posting ke Facebook dan Instagram, membalas komentar, dan memantau iklan kamu — semuanya dari satu halaman ini. Bayangkan seperti asisten robot yang bekerja untukmu 24 jam!
Apa saja yang bisa dilakukan?
✍️

Buat & Posting Otomatis

Robot akan membuat tulisan postingan sendiri menggunakan AI, lalu langsung mempostingnya ke Facebook dan Instagram kamu.

💬

Balas Komentar Otomatis

Kalau ada orang yang berkomentar di postinganmu, robot ini akan membaca komentar itu dan membalasnya secara otomatis dengan jawaban yang sopan dan relevan.

📊

Pantau Performa Iklan

Kamu bisa melihat data iklan Facebook kamu — berapa orang yang melihat, mengklik, dan berapa biayanya.

⚙️

Atur Semua Pengaturan

Ubah token akses, jadwal posting, topik konten, dan berbagai pengaturan lainnya langsung dari halaman ini tanpa perlu edit file.

🔌 Status Bar — pojok kanan atas
Di pojok kanan atas ada dua tombol kecil yang menunjukkan apakah aplikasi sudah terhubung ke akun Facebook dan Instagram kamu.
Arti warna lampu
●

Lampu Hijau = Terhubung ✅

Bagus! Aplikasi sudah berhasil masuk ke akun Facebook/Instagram kamu. Semua fitur siap digunakan.

●

Lampu Merah = Tidak Terhubung ❌

Ada masalah koneksi. Biasanya karena token akses yang salah atau kadaluarsa. Pergi ke tab ⚙️ Pengaturan dan periksa pageAccessToken.

●

Lampu Abu = Belum Dikonfigurasi ⬜

Fitur ini belum diatur. Untuk Instagram, kamu perlu mengisi igUserId dan igAccessToken di pengaturan.

Cara memperbaiki jika merah
1

Pergi ke Meta Developer

Buka developers.facebook.com → pilih aplikasimu → Tools → Graph API Explorer.

2

Salin token baru

Generate token baru dengan permission pages_manage_posts dan pages_read_engagement, lalu salin tokennya.

3

Tempel di Pengaturan

Buka tab ⚙️ Pengaturan di bawah halaman ini, cari pageAccessToken, ganti nilainya dengan token baru, lalu klik Save Config.

✍️ Tombol "Post Now"
Fitur ini meminta AI untuk membuat tulisan postingan secara otomatis, lalu langsung mempublikasikannya ke Facebook Page (dan Instagram jika dikonfigurasi). Proses ini bisa memakan waktu 5–15 detik.
Langkah-langkah
1

Pilih Jenis Postingan (opsional)

Di kotak dropdown sebelum tombol "Post Now", kamu bisa memilih jenis postingan:

🔄 Auto — robot pilihkan otomatis secara bergantian
📣 Promo — postingan promosi produk/diskon
⭐ Testimonial — cerita pelanggan yang puas
🎯 Product Highlight — highlight fitur produk
📚 Educational — konten edukasi/tips

💡 Pilih "Auto" jika kamu mau biarkan robot yang memilih supaya konten bervariasi.
2

Klik tombol "▶ Post Now"

Tombol biru di baris aksi. Setelah diklik, tombol akan berubah jadi "Running…" dan kamu harus menunggu.

3

Tunggu notifikasi

Kalau berhasil, akan muncul kotak hijau di pojok kanan bawah bertuliskan "Post published!". Postingan baru akan langsung muncul di panel "Recent Posts".

⚠️ Kalau gagal, periksa apakah status FB sudah hijau dan token masih valid.
Cara mengubah topik/produk postingan
⚙️

Edit di Config Editor

Scroll ke bawah ke bagian Config Editor. Cari bagian "businessContext" dan ubah deskripsi bisnis, produk, dan target audiensnya sesuai kebutuhanmu.

💬 Tombol "Check Comments"
Fitur ini akan memeriksa semua komentar terbaru di postingan Facebook kamu, lalu secara otomatis membalas komentar yang belum dibalas menggunakan AI. Balasannya akan terasa natural dan sesuai konteks komentar.
Cara menggunakan
1

Pastikan koneksi FB hijau

Lihat status bar di pojok kanan atas. Pastikan lampu FB berwarna hijau sebelum lanjut.

2

Klik "✕ Check Comments"

Tombol abu-abu di baris aksi. Robot akan mulai membaca semua komentar di postingan terbaru kamu.

3

Pantau di Live Logs

Di panel Live Logs sebelah kiri, kamu bisa melihat proses robot sedang membalas komentar apa. Setiap balasan akan tercatat di sana.

💡 Robot hanya membalas komentar yang BELUM pernah dibalas sebelumnya. Sudah dibalas = dilewati.
Pengaturan otomatis (jadwal)
⏰

Jadwal auto reply

Di Config Editor, cari "commentReply" → "schedule" untuk mengatur seberapa sering robot cek komentar secara otomatis (format cron, contoh: "*/30 * * * *" = setiap 30 menit).

📊 Tombol "Check Ads"
Fitur ini mengambil data performa iklan dari Facebook Ads Manager kamu dan menampilkannya dalam tabel yang mudah dibaca. Kamu bisa melihat berapa banyak orang yang melihat iklan, mengklik, dan berapa biayanya.
Cara menggunakan
1

Klik "▶ Check Ads"

Tombol hijau di baris aksi. Robot akan menghubungi Facebook API untuk mengambil data iklan terbaru.

2

Lihat tabel Ad Metrics

Scroll ke bawah ke bagian Ad Metrics (latest). Tabel akan memperlihatkan data iklan dengan kolom berikut:

📅 Date — tanggal data
👁️ Impressions — berapa kali iklan dilihat
👥 Reach — berapa orang unik yang melihat
🖱️ Clicks — berapa kali diklik
💰 Spend — total biaya iklan (USD)
📈 CTR — persentase yang mengklik dari yang melihat

3

Refresh kapan saja

Klik tombol Refresh kecil di pojok kanan panel Ad Metrics untuk memperbarui data tanpa harus scroll ke atas.

⚠️ Pastikan kamu sudah mengisi adAccountId di Config Editor agar fitur ini berfungsi.
Syarat agar fitur Ads berfungsi
✔

Yang perlu diisi di Config

• adAccountId — ID akun iklan (format: act_123456789)
• pageAccessToken — harus punya permission ads_read

⚙️ Config Editor — bagian paling bawah
Config Editor adalah tempat kamu mengatur semua pengaturan aplikasi. Formatnya berupa JSON — seperti daftar pengaturan dengan nama dan nilainya. Kamu tidak perlu jago coding untuk mengeditnya!
Cara mengedit pengaturan
1

Scroll ke bawah halaman

Config Editor ada di bagian paling bawah halaman, dengan latar hitam dan tulisan berwarna hijau.

2

Klik di area teks dan edit

Cari bagian yang ingin diubah. Misalnya untuk mengubah token Facebook, cari tulisan "pageAccessToken" lalu ganti nilai di dalam tanda kutip setelah tanda :.

💡 Jangan hapus tanda { }, [ ], tanda kutip " ", atau koma , — formatnya harus tetap rapi!
3

Klik "Save Config"

Tombol biru di pojok kanan bawah Config Editor. Kalau berhasil, akan muncul notifikasi hijau "Config saved!".

4

Restart agent (jika perlu)

Beberapa pengaturan (seperti jadwal posting) baru berlaku setelah kamu me-restart server. Lihat pesan di bawah tombol Save untuk info lebih lanjut.

Pengaturan penting yang sering diubah
🔑

Token & ID Akun

• pageAccessToken — Token Facebook Page kamu
• pageId — ID halaman Facebook kamu
• adAccountId — ID akun iklan (act_...)
• igUserId + igAccessToken — untuk Instagram

📝

Informasi Bisnis

• businessContext.description — Ceritakan bisnis kamu agar AI bisa buat konten yang tepat
• businessContext.products — Daftar produk/layanan kamu
• businessContext.targetAudience — Siapa target pembeli kamu

⏱️

Jadwal Otomatis

• posting.schedule — Kapan posting otomatis berjalan
• commentReply.schedule — Kapan cek komentar otomatis

Format jadwal: "0 9 * * *" = setiap hari jam 9 pagi
"*/30 * * * *" = setiap 30 menit

📋 Panel "Live Logs" — kiri tengah
Live Logs adalah "jendela" untuk melihat apa yang sedang dilakukan robot secara real-time. Setiap kali robot melakukan sesuatu — posting, membalas komentar, mengambil data — akan muncul catatan di sini.
Arti warna teks log
i

Abu-abu — INFO

Informasi biasa. Robot sedang berjalan normal. Tidak ada yang perlu dikhawatirkan.

✓

Hijau — SUCCESS

Berhasil! Postingan berhasil dipublikasikan, komentar berhasil dibalas, dsb.

!

Kuning — WARN

Peringatan. Ada sesuatu yang kurang sempurna tapi robot masih bisa berjalan. Perlu diperhatikan.

✕

Merah — ERROR

Ada kesalahan. Biasanya berarti token kadaluarsa, koneksi internet bermasalah, atau konfigurasi salah. Baca pesannya untuk tahu penyebabnya.

Tips penggunaan
🗑️

Tombol "Clear"

Klik tombol "Clear" di pojok kanan atas panel Log untuk membersihkan tampilan. Log lama di server tidak ikut terhapus, hanya tampilan yang dibersihkan.

📡

Log muncul otomatis

Log terhubung secara langsung (real-time) ke server. Kalau sambungan terputus, akan muncul pesan "Log stream disconnected" dan otomatis sambung kembali dalam 3 detik.

⚠️
Facebook tidak terhubung — kemungkinan token salah atau kadaluarsa
Error 404 pada posting atau 400 pada /posts biasanya berarti token yang dipakai adalah User Access Token, bukan Page Access Token. Klik tombol di bawah untuk melihat daftar halaman yang bisa diakses token kamu.
